Project Details

As part of my continuous journey to deepen my knowledge in the field of cybersecurity, I completed the Web Fundamentals in Cybersecurity course offered by TryHackMe. This course provided a thorough introduction to the core concepts of web security, covering a wide range of topics including common vulnerabilities, attack vectors, and the essential security measures needed to protect web applications. Through hands-on labs and real-world scenarios, I gained practical experience in identifying and mitigating security risks. The course extensively covered key vulnerabilities such as SQL injection, cross-site scripting (XSS), and cross-site request forgery (CSRF), among others.
It also introduced defensive strategies, teaching me how to implement secure coding practices, protect web servers, and understand the security mechanisms that can safeguard users from potential threats. By the end of the course, I had a solid understanding of how web applications are attacked and how to defend them, which significantly enhanced my cybersecurity skills.
